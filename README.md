## LangChain Project

### 🔹 [AI Study Evaluator](https://github.com/AtharvNiprul74/Deep-Learning-Lab-Codes/tree/main/LangChain%20Project)

A Streamlit-based application that uses **LangChain + Gemini (LLM)** to evaluate and improve student answers.

- Provides feedback, missing points, and scoring  
- Suggests follow-up questions for better understanding  
- Maintains conversation memory using `ConversationBufferMemory`  
- Supports two modes: *Evaluate Answer* and *Improve Answer*
